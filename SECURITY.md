# Security Policy

## Reporting Issues

Please report security issues through GitHub security advisories when available, or open a minimal issue without exploit details.

## Scope

`runbook-lint` parses local Markdown and writes reports. It should not execute runbook commands or contact infrastructure. Bugs that execute commands, write outside requested output paths, or produce misleading security-sensitive results are in scope.

## Supported Versions

The current `0.x` line receives security fixes.
